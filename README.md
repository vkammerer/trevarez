Borne tactile pour l'exposition permanente "Bâtir un rêve" au château de Trévarez.

Visible [ici pour l'instant](http://www.vincent-kammerer.com/trevarez/)

### Installation
```
npm i
```

### Development
```
npm run dev
```

### Web Build
```
npm run build
```

### App Build
OSX:
```
npm run build:osx
```

Windows
```
npm run build:win
```

### Deployment
```
npm run deploy
```
