### Installation
```
npm i
```

### Development
```
npm run dev
```

### Web Build
```
npm run build
```

### App Build
OSX:
```
npm run build:osx
```

Windows
```
npm run build:win
```

### Deployment
```
npm run deploy
```

### Release installer
```
npm version [major|minor|patch]
git push && git push --tag
```
